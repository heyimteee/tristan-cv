# Tristan CV

A polished and fully responsive static CV website built with Vue 3 + Vite. The project presents Tristan Al Harrish Basori's profile, education, experience, selected projects, and technical skills with a modern visual style, smooth entrance animations, and consistent interactions across desktop and mobile.

## Live Demo

Will be added after Vercel deployment.

## Features

- Professional one-page CV layout with strong visual hierarchy
- Hero section with personal profile image (3:4 ratio) and quick contact links
- Education timeline with scroll-triggered entrance animation
- Experience and project cards with staggered reveal and hover interactions
- Technical skills section with matching reveal and hover effects
- Custom abstract brand logo used as site logo and favicon
- Mobile-first responsive behavior for small phones through large desktops

## Tech Stack

- Vue 3
- Vite
- Vanilla CSS (custom design system + animations)

## Getting Started

### Prerequisites

- Node.js 20+
- npm 11+

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Deployment

This project is optimized for Vercel:

- Framework Preset: `Vite`
- Build Command: `npm run build`
- Output Directory: `dist`
